# True Victory — Financial Section POC

A self-contained proof-of-concept of the **Financial Section** (Increment 3) for
True Victory's CFO, built on the VODS design system (dark theme, True Gold accent,
self-hosted "Good Times" display face).

The entire app — React, the application code, fonts, and images — is bundled into a
single `index.html` with no external or CDN dependencies. Open it in any current
browser (Chrome, Edge, Safari 16.4+, Firefox 113+).

**Live:** https://slymasterfunk.github.io/true-victory-financial-poc/
